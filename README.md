# 음식 사진 칼로리 예측 앱

## 프로젝트 개요
본 프로젝트는 음식 사진을 입력으로 받아 딥러닝 모델을 통해 음식 종류를 분류하고,
해당 음식의 예상 칼로리를 사용자에게 제공하는 Android 애플리케이션이다.

## 사용 기술
- Android Studio (Kotlin, Jetpack Compose)
- TensorFlow Lite
- CNN 기반 사전 학습 이미지 분류 모델 (MobileNet 계열)
- Coil (이미지 로딩)

## 주요 기능
- 갤러리에서 음식 사진 선택
- 딥러닝 모델을 통한 음식 이미지 분석
- 음식 이름 및 예상 칼로리 출력

## 기대 효과
사용자는 음식 섭취 전 간단히 칼로리를 예측하여
건강한 식습관 관리에 도움을 받을 수 있다.
