# 음식 사진 기반 칼로리 예측 Android 앱

## 1. 프로젝트 개요
본 프로젝트는 딥러닝 기술을 활용하여 음식 사진을 분석하고,
해당 음식의 종류 및 예상 칼로리를 사용자에게 제공하는 Android 애플리케이션이다.

최근 스마트폰 사용 증가와 함께 건강 관리에 대한 관심이 높아지고 있으며,
특히 음식 섭취 시 칼로리 정보를 직관적으로 확인할 수 있는 서비스의 필요성이 커지고 있다.
이에 본 프로젝트는 사용자가 음식 사진을 촬영하거나 선택하면
딥러닝 모델을 통해 자동으로 분석 결과를 제공하는 앱을 구현하는 것을 목표로 한다.

---

## 2. 프로젝트 목적
- 딥러닝 기반 이미지 분류 모델을 실제 모바일 애플리케이션에 적용
- Android 환경에서 TensorFlow Lite를 활용한 추론 과정 이해
- 인공지능 기술을 활용한 실생활 응용 앱 개발 경험

---

## 3. 시스템 구성 및 동작 방식
본 애플리케이션은 다음과 같은 흐름으로 동작한다.

1. 사용자가 갤러리에서 음식 사진을 선택
2. 선택된 이미지를 Bitmap 형태로 변환
3. TensorFlow Lite 기반 딥러닝 모델에 입력
4. 음식 이미지 분류 결과 출력
5. 분류된 음식에 대응하는 예상 칼로리 정보 표시

전체 과정은 모바일 환경에서도 빠르게 실행될 수 있도록
경량화된 딥러닝 모델을 사용하는 구조로 설계되었다.

---

## 4. 사용 기술 및 개발 환경

### 개발 환경
- Android Studio
- Kotlin
- Jetpack Compose (UI 구성)

### 딥러닝 및 AI 기술
- TensorFlow Lite
- CNN(Convolutional Neural Network) 기반 사전 학습 이미지 분류 모델
- MobileNet 계열 경량 모델 구조 참고

### 기타 라이브러리
- Coil: 이미지 로딩 및 표시

---

## 5. 딥러닝 모델 설명
본 프로젝트에서는 모바일 환경에서의 효율적인 추론을 위해
TensorFlow Lite 기반의 사전 학습된 CNN 모델을 사용하였다.

CNN은 이미지 특징을 효과적으로 추출하는 데 적합한 구조로,
음식 이미지 분류와 같은 컴퓨터 비전 문제에서 널리 사용된다.
특히 MobileNet 계열 모델은 연산량이 적고 성능 대비 효율이 높아
스마트폰 환경에 적합한 딥러닝 모델이다.

이번 과제에서는 모델의 직접 학습보다는
모바일 앱과 딥러닝 모델의 연동 구조를 이해하고 구현하는 데 초점을 두었다.

---

## 6. 주요 기능
- 음식 사진 선택 기능
- 딥러닝 모델을 이용한 음식 이미지 분석
- 음식 이름 및 예상 칼로리 결과 출력
- 직관적인 UI를 통한 사용자 경험 제공

---

## 7. 한계점 및 개선 방향
- 현재는 제한된 음식 분류 결과를 기반으로 칼로리를 예측함
- 실제 칼로리 값은 음식의 양과 조리 방식에 따라 차이가 있을 수 있음

향후 개선 방향으로는
- 더 다양한 음식 데이터셋을 활용한 모델 재학습
- 음식 양 추정 기능 추가
- 서버 기반 모델 연동을 통한 성능 향상
등을 고려할 수 있다.

---

## 8. 기대 효과
본 애플리케이션을 통해 사용자는
음식 섭취 전 간단한 사진 입력만으로 칼로리 정보를 확인할 수 있으며,
이는 건강한 식습관 형성과 자기 관리에 도움을 줄 수 있다.

또한 본 프로젝트는
딥러닝 이론을 실제 Android 애플리케이션에 적용해보는 실습 사례로서
교육적 측면에서도 의미를 가진다.
